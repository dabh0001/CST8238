  1. What are the different parts of a CSS style rule?
  2. What is the difference between a relative and absolute measure unit in CSS? Why are relative units preferred over absolute units in CSS?
  3. What are an element selector and grouped element selector in CSS? Provide an example of each.
  4. What are class selectors in CSS? What are id selectors in CSS?
  5. What are contextual selectors in CSS?

  1. A style rule is made of three parts: selector, property and value.
  2. Relative length units specify a length relative to another length property; absolute length units are fixed and a length expressed in any of these will appear as exactly that size. Beacuse Relative length units scales better between different rendering mediums.
  3. The element selector selects elements based on the element name. For example, you can select all 'p' element like this, p {color: red; font-size: 10px; } If you have elements with the same style definitions, you can group the selectors, to minimize the code, like this, h1, h2, p {color: blue; text-align: center;}
  4. A class selector is a name preceded by a full stop . and an id selector is a name preceded by a hash character #.
  5. "Contextual selectors" allow you to specify different styles for different parts of your document. You can assign styles directly to specific HTML tags, or you can create independent classes and assign them to tags in the HTML.
