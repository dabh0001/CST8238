1. What are the different parts of a CSS style rule?

There are two parts of a CSS style rule: the selector and the delcaration. The selector is the HTML element you want to style, and the delcaration block contains one or more delcarations separated by semicolons.

2. What are an element selector and grouped element selector in CSS? Provide an example of each.

The element selector is a pattern used to select the element you want to style. An example of an element selector is:

p{

background-color: red;

}

The grouped element selector in CSS are when several selectors shared the same declarations; they can be grouped together to save writing the same rule more than once. An example of an group element selector is:

h1, h2, h3, h4 {

font-size: 20px;

}

3. What are class selectors in CSS? What are id selectors in CSS?

Class selectors and id selectors are a way to define your selectors. In CSS, is identified by having a . before their name, while an id selector would have a hash tag before their name. In CSS a class selector would look like this:

#container{

padding: 10px;

}

and an id selector would look like this:

.footer{

width: 10px;

}

4. What are contextual selectors in CSS?

The contextual selectors in the CSS allows you to specificy different styles for different parts of your document. You can assign styles directly to specific HTML tags, or you can create independent class and assign them to tags in the HTML.

table p {

property: value;

}

5. What are the three cascade principles used by browsers when style rules conflict?

The three cascade principles used by browsers when style rules conflict is inheritence, cascasde and specificity.

Inheritence: Some styles are inherited down the HTML documentation tree, while others are not.

Specificity: CSS rules often conflict with one another. The trick is understanding how conlficting rules will apply

Cascade: Style sheets are cascade by order of importance. If rules in different style sheets conflict with one another, th rule from the most important style sheet wins.
